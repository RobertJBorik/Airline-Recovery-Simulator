# Airline Recovery Simulator

An event-driven airline operations simulator built in Python.
Live Demo: https://airline-recovery-simulator.streamlit.app/

## Features

- Aircraft routing and rotations
- Event-driven simulation engine
- Monte Carlo simulation
- Weather, gate, and maintenance disruptions
- Delay propagation
- Streamlit analytics dashboard

## Planned Features

- Aircraft swap recovery
- Reserve aircraft deployment
- Multi-day scheduling
- Simulation replay
